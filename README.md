# Route Helper Monorepo

Environment scaffold for:
- `apps/mobile` (Expo React Native)
- `apps/api` (Fastify API)
- `apps/worker` (scheduled job runner)
- `packages/shared` (shared types and validation)

## Quick start

1. Install Bun.
2. Run `bun install`.
3. Start each app:
   - `bun run dev:mobile`
   - `bun run dev:api`
   - `bun run dev:worker`

## Notes

This is a setup-only baseline. Domain functionality is intentionally not implemented yet.
